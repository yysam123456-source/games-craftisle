# Minesweeper - Complete Game Guide

## 🎮 Game Overview

Minesweeper is the ultimate logic puzzle that has frustrated and fascinated players for decades. Originally created in the 1960s, it gained worldwide fame as a bundled game with Microsoft Windows (1990-200, appearing on over 500 million computers. The concept is simple yet brilliant: clear a minefield using logic and deduction, with numbers revealing how many mines are adjacent to each cell. One wrong click and—BOOM!—game over. It's chess meets gambling, requiring both logical precision and calculated risk-taking.

## 📜 History & Cultural Impact

- **1960s**: First Minesweeper-like game created (Mine探测器)
- **1989**: Robert Donier creates "Minesweeper" for Windows 3.0 (bundled free)
- **1992**: Minesweeper officially included in Windows 3.1 (over 500 million copies)
- **1990s**: Becomes the most-played computer game in history
- **2000s**: Web and mobile versions proliferate
- **2015**: Minesweeper World Championship established
- **Legacy**: Inspired countless puzzle games; considered one of the greatest puzzle games ever

## 🎯 How to Play

### Basic Rules
1. **The Grid**: Minefield with hidden mines scattered throughout
2. **The Goal**: Reveal all non-mine cells without detonating any mines
3. **Numbers**: Revealed cells show how many mines are in adjacent cells (0-8)
4. **Flags**: Mark suspected mine locations with flags
5. **Winning**: Clear all non-mine cells; losing: click on a mine

### Controls
- **Left Click**: Reveal a cell
- **Right Click**: Place/remove a flag
- **Middle Click/Both Buttons**: Chord (reveal adjacent cells if correct number of flags placed)
- **Touch**: Tap to reveal; long press to flag

### Difficulty Levels
- **Beginner**: 9x9 grid, 10 mines
- **Intermediate**: 16x16 grid, 40 mines
- **Expert**: 30x16 grid, 99 mines
- **Custom**: Create your own challenge

## 🧠 Strategy Guide

### Beginner Techniques

#### 1. Start in a Corner
- First click is always safe (game regenerates if you hit a mine)
- Corners often reveal large open areas

#### 2. Read the Numbers
- **1**: Exactly one mine in the 8 adjacent cells
- **2**: Exactly two mines in adjacent cells
- **3-8**: Corresponding number of mines nearby

#### 3. Flag with Confidence
- Only flag when you're 100% sure a cell contains a mine
- Unnecessary flagging wastes time

#### 4. Use "1-1" Patterns
- If you see "1" next to another "1" with overlapping unknown cells—the mine must be in the overlap
- This pattern appears frequently and is easy to miss

### Intermediate Techniques

#### 5. The "One-Mine-Remaining" Rule
- If a number cell has exactly N adjacent unknown cells and N adjacent flags—all other adjacent cells are safe to reveal
- Conversely: if a number cell needs N more mines and has exactly N unknown cells—all those unknowns are mines!

#### 6. Pattern Recognition
- **1-2-1 Pattern**: Mines are at positions 1 and 3 (the "1" cells)
- **1-2-2-1 Pattern**: Mines are at positions 2 and 3
- These patterns appear constantly—learn to spot them instantly

#### 7. Edge Strategies
- Edges and corners have fewer adjacent cells—easier to deduce
- Use edges to narrow down mine locations

### Advanced Techniques

#### 8. Probability Calculation
- When logic isn't enough, calculate probabilities
- Example: Two unknown cells, one mine—50% chance each
- Sometimes you MUST guess—minimize risk by choosing lowest probability

#### 9. The "Temple" Strategy
- Advanced pattern involving 3-2-3 or similar configurations
- Requires visualizing multiple scenarios
- Used by expert players to solve expert puzzles faster

#### 10. Speed Techniques
- **Mouse Movement**: Minimize hand movement between cells
- **Keyboard Shortcuts**: Some versions allow keyboard navigation
- **Chording**: Use middle-click to instantly reveal multiple cells

### Common Mistakes
- Clicking without counting adjacent flags
- Flagging based on guesswork instead of logic
- Not using the "one-mine-remaining" rule
- Panic-clicking when the grid gets crowded
- Missing obvious number patterns
