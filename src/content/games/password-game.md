# The Password Game - Complete Guide

## 🎮 Game Overview

**The Password Game** is a hilarious, frustrating, and oddly addictive password creation game. What starts as a simple "create a password" task quickly evolves into a linguistic nightmare as increasingly absurd rules are added one by one.

Inspired by the original [The Password Game by Neal Agarwal](https://neal.fun/password-game/), this game will test your creativity, patience, and ability to think outside the box!

## 🎯 How to Play

### Objective
Create a password that satisfies **ALL** the rules, simultaneously.

### Rules Progression
New rules appear one by one as you meet the current ones. Here's what you'll face:

| Rule # | Requirement | Difficulty |
|--------|-------------|------------|
| 1 | At least 8 characters | ⭐ Easy |
| 2 | At least one uppercase letter | ⭐ Easy |
| 3 | At least one lowercase letter | ⭐ Easy |
| 4 | At least one number | ⭐ Easy |
| 5 | At least one special character (!@#$%^&*) | ⭐ Medium |
| 6 | No spaces | ⭐ Easy |
| 7 | At least one emoji | ⭐ Medium |
| 8 | Contains a palindrome | ⭐ Hard |
| 9 | Contains a Roman numeral | ⭐ Medium |
| 10 | Contains a chemical element symbol | ⭐ Hard |
| 11 | Contains leet speak (a→4, e→3, i→1, o→0) | ⭐ Medium |
| 12 | At least 20 characters | ⭐ Medium |

### Controls
- **Keyboard**: Type your password
- **Emoji Keyboard**: Use Win+. (Windows) or Cmd+Ctrl+Space (Mac) to insert emojis
- **Mobile**: Use touch keyboard with emoji panel

## 🏆 Strategies for Success

### Strategy 1: **The "Kitchen Sink" Approach**
Throw EVERYTHING into your password from the start:
```
P@ssw0rdI!HeLi3️⃣racecar4️⃣
```
- ✅ Has uppercase, lowercase, numbers, special chars
- ✅ Has emoji (3️⃣)
- ✅ Has Roman numeral (I, IV, etc.)
- ✅ Has chemical element (He, Li)
- ✅ Has palindrome (racecar)
- ✅ Has leet speak (a→@, o→0, e→3)

### Strategy 2: **Build Incrementally**
1. Start with a strong base: `P@ssw0rd`
2. Add requirements as they appear
3. Rearrange as needed to fit new rules

### Strategy 3: **The "Cheat Sheet" Method**
Keep a notepad with:
- Common palindromes: racecar, madam, level, kayak
- Roman numerals: I, V, X, L, C, D, M
- Chemical symbols: H, He, Li, Be, B, C, N, O, F, Ne
- Leet speak: a=4, e=3, i=1, o=0, s=5, t=7

## 💡 Pro Tips

### Tip 1: **Master the Palindrome**
A palindrome reads the same forwards and backwards. Easy ones:
- `racecar`
- `madam`
- `level`
- `kayak`
- `radar`
- `civic`

**Pro move**: Use a palindrome as your password's core!

### Tip 2: **Roman Numerals are Everywhere**
Common Roman numerals:
- I = 1
- V = 5
- X = 10
- L = 50
- C = 100
- D = 500
- M = 1000

**Tip**: Just add "I" or "V" somewhere in your password!

### Tip 3: **Chemical Elements for the Win**
Single-letter elements (easiest):
- H (Hydrogen)
- B (Boron)
- C (Carbon)
- N (Nitrogen)
- O (Oxygen)
- F (Fluorine)
- P (Phosphorus)
- S (Sulfur)
- K (Potassium)
- V (Vanadium)
- Y (Yttrium)
- I (Iodine)
- W (Tungsten)
- U (Uranium)

**Tip**: Most passwords already contain H, C, N, or O!

### Tip 4: **Emoji Placement Matters**
- Emojis count as characters but don't break rules
- Place them at the end to avoid interfering with other requirements
- Use number emojis (1️⃣2️⃣3️⃣) to double as actual numbers!

### Tip 5: **Watch Out for Rule Conflicts**
- Adding an emoji might break your "no spaces" rule if you're not careful
- Uppercase Roman numerals (I, V, X) count as uppercase letters
- Lowercase chemical symbols (he, li, be) count as lowercase letters

## 📊 Difficulty Curve

### Early Game (Rules 1-6)
- **Difficulty**: ⭐ Easy
- **Strategy**: Straightforward, any decent password works
- **Time**: 1-2 minutes

### Mid Game (Rules 7-9)
- **Difficulty**: ⭐⭐ Medium
- **Challenge**: Emoji + palindrome + Roman numeral
- **Strategy**: Start rearranging, get creative
- **Time**: 5-10 minutes

### Late Game (Rules 10-12)
- **Difficulty**: ⭐⭐⭐⭐ Hard
- **Challenge**: Chemical elements + leet speak + 20+ chars
- **Strategy**: Complete rewrite usually needed
- **Time**: 10-30 minutes

### End Game (All 12 Rules)
- **Difficulty**: ⭐⭐⭐⭐⭐ Impossible?
- **Challenge**: Keeping ALL rules satisfied simultaneously
- **Strategy**: The "Kitchen Sink" approach
- **Time**: However long it takes!

## 🤣 Funny Moments & Fails

### The "Almost Had It" Moment
You finally meet Rule 11 (leet speak), but in doing so, you accidentally remove the emoji that was satisfying Rule 7. Now you're scrambling to add it back without breaking anything else!

### The "Space" Trap
You add a Roman numeral "I", but accidentally type "I " (with a space), breaking Rule 6. Now you're frantically deleting spaces while trying not to mess up other rules.

### The "Length" Struggle
You're at 19 characters with all rules satisfied, but Rule 12 requires 20+. One more character and... oops, you broke the palindrome rule!
