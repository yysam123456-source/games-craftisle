# Infinite Craft

Combine elements to discover new items. Start with the four basic elements and see how far your creativity can take you!

## How to Play

1. Drag an element from the sidebar onto the workspace
2. Drag one element onto another to combine them
3. Discover new elements from the combinations
4. There are over 1200 elements to discover!
